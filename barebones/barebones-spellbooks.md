# Pelkistetyn laitoksen loitsukirjat

|         |                         |                                                                                                                                                                                                                                                                                                                                                            |
| ------- | ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1**   | **Kiinnitys**           | Esine peittyy erittäin tahmeaan limaan. _Viereiset esineet tarttuvat kirjaan todella tiukasti._                                                                                                                                                                                                                            |
| **2**   | **Ankkurointi**         | Käsistäsi pursuaa vahva lanka, joka kiinnittyy kahteen pisteeseen 15 metrin säteellä kummallekin puolellesi. _Jos köysi vedetään sen selkämyksen rautasilmukan läpi, siitä tulee yhtä painava kuin norsusta._                                                                                                              |
| **3**   | **Esineen elävöitys**   | Esine tottelee käskyjäsi parhaansa mukaan. _Muovautuvaa kuin savi. Lapsenomainen nauru pursuaa sen sivuilta._                                                                                                                                                                                              |
| **4**   | **Inhimillistys**       | Yksi eläin saa joko ihmisen älykkyyden tai ulkonäön yhden päivän ajaksi. _Vinkuu, kehrää ja murisee kohtelun mukaan._                                                                                                                                                                                                      |
| **5**   | **Loihtosilmä**         | Voit nähdä läpi maagisen leijuvan silmämunan, joka lentelee ympäriinsä käskystäsi. _Vaatii vesisuihkauksen avautuakseen._                                                                                                                                                                                                  |
| **6**   | **Astraalivankila**     | Esine jähmettyy ajassa ja avaruudessa haavoittumattoman kristallikuoren sisään. _Hiljaiset ja abstraktit kasvot huutavat tuskissaan sisältä._                                                                                                                                                                              |
| **7**   | **Puoleenveto**         | Kaksi esinettä vetävät toisiaan puoleensa voimakkaan magneettisesti, jos ne tulevat 3 metrin päähän toisistaan. _Lähellä olevat kompassit pyörivät vimmatusti._                                                                                                                                                            |
| **8**   | **Harhakuulo**          | Luot harhaääniä, jotka vaikuttavat tulevan valitsemastasi suunnasta. _Tuottaa satunnaisia ja toisinaan sopimattomia ääniä pitkin päivää_.                                                                                                                                                                                  |
| **9**   | **Höpötys**             | Olennon on toistettava äänekkäästi ja selvästi kaikki, mitä ajattelet. Muuten se on vaiti. _Kun tekstiä luetaan ääneen, muiden sanat muuttuvat epäselviksi._                                                                                                                                               |
| **10**  | **Syötinkukka**         | Maasta versoaa kasvi, joka huokuu mätänevän lihan hajua. _Houkuttelee kärpäsiä._                                                                                                                                                                                                                                           |
| **11**  | **Petomuoto**           | Sinä ja omaisuutesi muututte tavanomaiseksi eläimeksi. _Paksun turkin peittämä, sen reunoilla on pieniä hampaita._                                                                                                                                                                                                         |
| **12**  | **Hämmennys**           | Valitsemasi olento ei kykene muodostamaan uusia lähimuistikuvia loitsun keston ajan. _Sen sisältö vaihtaa paikkaa ja muuttuu joka kerta, kun se avataan._                                                                                                                                                                  |
| **13**  | **Kehonvaihdos**        | Vaihdat kehoa koskettamasi olennon kanssa. Jos toinen keho kuolee, niin myös toinen kuolee. _Kannessa on kuva viimeisimmästä olennosta, joka luki sen._                                                                                                                                                    |
| **14**  | **Lumous**              | Olento, jonka voit nähdä, kohtelee sinua kuin ystävää _Lämpimän tuntuinen ja tuoksuu ruusuilta._                                                                                                                                                                                                                                           |
| **15**  | **Käsky**               | Kohde tottelee yhtä kolmen sanan käskyä, joka ei aiheuta sille itselleen vahinkoa. Kapenee ajan myötä, kunnes lopulta katoaa lopullisesti._                                                                                                                                                           |
| **16**  | **Ymmärrys**            | Osaat lyhyen aikaa sujuvasti kaikkia kieliä. _Valuu kirjaimia, värjäten kaiken mihin se koskettaa._                                                                                                                                                                                                                        |
| **17**  | **Vaahtopurkaus**       | Kädestäsi suihkuaa tiheää vaahtoa, joka peittää kohteen. _Huokoisen pehmeä, kostea ja siinä on saippuajäämiä._                                                                                                                                                                                                             |
| **18**  | **Kasvien hallinta**    | Lähistöllä olevat kasvit ja puut tottelevat sinua ja saavat kyvyn liikkua hitaasti. _Lehtiä kasvaa pitkin selkääsi ja tuoksu on hieman mädäntynyt._                                                                                                                                                                        |
| **19**  | **Säiden hallinta**     | Voit muuttaa säätä mielesi mukaan, mutta et muuten hallitse sitä. _Hyvin vastustuskykyinen tuli- ja vesivahingolle._                                                                                                                                                                                                       |
| **20**  | **Haavojen parannus**   | Paranna 1d4 STR kerran päivässä olennolle, jota voit koskettaa. _Tuoksuu etikalle ja timjamille. Muuttuu punaiseksi käytön jälkeen._                                                                                                                                                                       |
| **21**  | **Kuuroutus**           | Kaikki lähellä olevat olennot kuuroutuvat. _Lähellä olevat soittimet soivat ajoittain, ikään kuin protestiksi._                                                                                                                                                                                                            |
| **22**  | **Taikuuden paljastus** | Voit nähdä tai kuulla lähellä olevia maagisia auroja. _Tuntuu lämpimältä, kun taikuutta käytetään lähellä._                                                                                                                                                                                                                |
| **23**  | **Irrottelu**           | Mikä vain ruumiinosasi voidaan halutessasi irrottaa ja kiinnittää uudelleen, aiheuttamatta kipua tai vahinkoa. Voit silti hallita niitä. _Kaikki repeytyneet tai turmeltuneet sivut uusiutuvat takaisin._                                                                                                  |
| **24**  | **Valeasu**             | Voit halutessasi muuttaa yhden hahmon ulkonäköä kunhan se pysyy humanoidina. Yritykset jäljitellä muita hahmoja vaikuttavat epätodellisilta. _Pinta on kirkas peili._                                                                                                                                      |
| **25**  | **Siirtymä**            | Esine näyttää olevan enintään 5 metrin päässä todellisesta sijainnistaan. _Joskus sen sisään on tungettu narunpätkiä, vaatteita ja lehtiä._                                                                                                                                                                                |
| **26**  | **Maanjäristys**        | Maa alkaa täristä rajusti. Rakennelmat voivat vaurioitua tai romahtaa. _Hiekkaa valuu kirjan nurkista, ilmeisesti taukoamatta._                                                                                                                                                                            |
| **27**  | **Notkeus**             | Kehosi voi venyä jopa 3 metriseksi. _Tuoksuu toffeelle ja on hyvin joustavaa._                                                                                                                                                                                                                                             |
| **28**  | **Elementaalivalli**    | Maasta nousee suora jää- tai tuliseinä, joka on 15 metriä pitkä ja 3 metriä korkea. _Iho ja lämpimämmät pinnat tarttuvat siihen käytön jälkeen._                                                                                                                                                                           |
| **29**  | **Näpistys**            | Näkyvä esine teleporttaa käsiisi. _Liittolaisen arvokasta omaisuutta saattaa joskus löytyä piilossa sen kansien välistä_.                                                                                                                                                                                                  |
| **30**  | **Kalankeuhko**         | Kohde voi hengittää veden alla kunnes se nousee jälleen pintaan. _Tuoksuu vahvasti mereltä. Houkuttelee villieläimiä._                                                                                                                                                                                     |
| **31**  | **Soihdutus**           | Kirkas energiapallo sinkoaa taivaalle valojuovan, joka paljastaa sijaintisi niin ystäville kuin viholliselle. _Hehkuu himmeästi täysin pimeässä_.                                                                                                                               |
| **32**  | **Sumupilvi**           | Tiheä sumu levittäytyy luotasi. _Veteen upotettuna kirja saa lopulta kaiken nesteen muuttumaan höyryksi._                                                                                                                                                                                                                  |
| **33**  | **Raivo**               | Lähellä oleva olento raivostuu väkivallan vimmassa. _Karkea, hiekkapaperinen kansi, joka tuhoaa minkä tahansa kirjan, johon se koskettaa._                                                                                                                                                                                 |
| **34**  | **Portti**              | Portti satunnaiseen maailmansfääriin avautuu. _Keskellä on suuri kaiverrettu reikä, joka päättyy tyhjyyteen. Sinne pudotettuja esineitä ei nähdä enää koskaan_.                                                                                                                                            |
| **35**  | **Gravitaatiosiirtymä** | Voit muuttaa painovoiman suuntaa, mutta vain itsellesi. _Kiinnittyy suurimpaan lähellä olevaan esineeseen._                                                                                                                                                                                                                |
| **36**  | **Ahneus**              | Olento tuntee ylivoimaista halua omistaa valitsemasi näkyvä esine. _Kansi vaihtuu omistajansa mukaan, vihjaillen hienovaraisesti hänen syvimmistä toiveistaan._                                                                                                                                                            |
| **37**  | **Nopeutus**            | Liikkumisnopeutesi kolminkertaistuu. _Sivut kääntyvät villisti avattuna. Voi aiheuttaa paperihaavoja._                                                                                                                                                                                                     |
| **38**  | **Vihanpito**           | Olento tuntee syvää vihaa toista olentoa tai ryhmää kohtaan ja haluaa tuhota ne. _Pitkäkestoinen altistuminen kirjalle voi aiheuttaa epäluuloa, vainoharhaisuutta ja epäluottamusta muita kohtaan._                                                                                                                        |
| **39**  | **Kuiskeiden kuulo**    | Voit kuulla heikot äänet selvästi. _Lukijan ääni voimistuu lyhyeksi aikaa jälkikäteen._                                                                                                                                                                                                                                    |
| **40**  | **Leijailu**            | Esine leijuu vaivattomasti puoli metriä maanpinnan yläpuolella. Se voi kantaa enintään yhden humanoidin. _Leijuu jos pudotetaan._                                                                                                                                                                          |
| **41**  | **Hypnotisointi**       | Olento menee transsiin ja vastaa totuudenmukaisesti yhteen sille esitettyyn kyllä- tai ei-kysymykseen. _Silmäänpistävät, pyörivät spiraalit koristavat sen kansia._                                                                                                                                                        |
| **42**  | **Hyytävä kosketus**    | Paksu jääkerros leviää kosketulle pinnalle, enintään 3 metrin säteellä. _Hanskat tarpeelliset. Ei syty tuleen_.                                                                                                                                                                                            |
| **43**  | **Omistajan tunnistus** | Koskettamasi esineen päälle ilmestyy kirjaimia, jotka kertovat esineen omistajan nimen, jos sellainen on olemassa. _Kirjan sisäkansissa lukee sen edellisen omistajan nimi._                                                                                                                                               |
| **44**  | **Valaistus**           | Leijuva valo liikkuu käskyjesi mukaan. _Kun sivuja pidetään valoa vasten, ne heijastavat värikkäitä sateenkaaria prismana._                                                                                                                                                                                                |
| **45**  | **Piilevä side**        | Kahta enintään 3 metrin etäisyydellä toisistaan olevaa esinettä ei voi siirtää yli 3 metrin päähän toisistaan. _Sivuja ei ole kiinnitetty liimalla tai langalla, mutta ne pysyvät silti yhdessä._                                                                                                                          |
| **46**  | **Kolkutus**            | Lähellä oleva tavallinen tai maaginen lukko aukeaa äänekkäästi. _Lukittu. Uusi omistaja "tuottaa" avaimen seuraavan ateriansa jälkeen._                                                                                                                                                                    |
| **47**  | **Loikkaus**            | Hyppäät enintään 3 metrin korkeuteen, kerran. _Kun se heitetään, se vain jatkaa lentoaan._                                                                                                                                                                                                                                 |
| **48**  | **Nesteilma**           | Ilma ympärilläsi muuttuu uimakelpoiseksi. _Leijuu omasta tahdostaan ja kimpoaa kaikesta, mihin se koskettaa._                                                                                                                                                                                                              |
| **49**  | **Taikavaimennus**      | Kaikkien lähellä olevien maagisten vaikutusten voima puolittuu. _Muinaisesineitä, jotka ovat 30 metrin sisällä loitsukirjasta ei voi ladata uudelleen._                                                                                                                                                                    |
| **50**  | **Pappila**             | Tukeva, kalustettu maalaistalo ilmestyy tunneiksi. Voit sallia ja kieltää pääsyn sinne tahtosi mukaan. _Jos se jätetään sisälle, niin kirja kuin mökki katoavat ikuisiksi ajoiksi._                                                                                                                        |
| **51**  | **Kuulahuuma**          | Taskusi ovat täynnä marmorikuulia ja ne täyttyvät uudelleen 30 sekunnin välein. _Tönäistäessä antaa leikkisän kolisevan äänen._                                                                                                                                                                                            |
| **52**  | **Naamioitus**          | Jonkin hahmon ulkonäkö ja ääni muuttuvat samanlaisiksi kuin koskettamallasi hahmolla. _Pitkään jatkunut käyttö aiheuttaa omistajalleen tiedostamattomia, mutta havaittavia elohiiriä._                                                                                                                                     |
| **53**  | **Pienennys**           | Koskettamasi olento kutistuu hiiren kokoiseksi. _Teksti on naurettavan ja koomisen suurta._                                                                                                                                                                                                                                |
| **54**  | **Kaksoskuvajainen**    | Sinusta ilmestyy harhakuvamainen kopio, joka on hallinnassasi. _Ajan saatossa omistaja alkaa pohtimaan, kuka on alkuperäinen ja kuka kaksoiskappale._                                                                                                                                                                      |
| **55**  | **Peilivaellus**        | Jokin peili muuttuu portaaliksi toiseen tänään katsomaasi peiliin. _Ei avaudu, ellei omistaja koputa kohteliaasti kanteen._                                                                                                                                                                                                |
| **56**  | **Monikätisyys**        | Saat hetkellisesti ylimääräisen käden. _Käytön jälkeen loitsija kärsii päivän ajan aavekivusta._                                                                                                                                                                                                                           |
| **57**  | **Yösfääri**            | Eteesi ilmestyy 15 metriä leveä pimeyden pallo, joka kuvastaa yötaivasta. _Sen pinnalla näkyy huomattava tähtikuvio_.                                                                                                                                                                                                      |
| **58**  | **Esinemuutos**         | Muutut miksi tahansa elottomaksi esineeksi, joka on kooltaan pianon ja omenan väliltä. _Omistajalla näkee kasvoja kaikkialla käytön jälkeen päivien ajan._                                                                                                                                                                 |
| **59**  | **Limamuoto**           | Muutut eläväksi hyytelöksi. _Siitä valuu hitaasti happoa, joka lopulta syö kaiken, mihin se koskettaa._                                                                                                                                                                                                                    |
| **60**  | **Tyynnytys**           | Lähelläsi oleva olento alkaa vieroksua väkivaltaa. _Tuoksuu jasmiinilta ja suitsukkeelta. Houkuttelee lapsia._                                                                                                                                                                                             |
| **61**  | **Kulku**               | Luo tilapäisen polun puun, kiven tai tiilen läpi. _Kirjan päälle pudotettu esine putoaa väistämättä kirjan läpi toiselle puolelle._                                                                                                                                                                                        |
| **62**  | **Kammotus**            | Lähelläsi oleva olento alkaa kammota valitsemaasi esinettä. _Ajan myötä sen sivut alkavat täyttyä ahdistavasta, abstraktista taiteesta._                                                                                                                                                                                   |
| **63**  | **Monttu**              | Maahan aukenee 3 metriä leveä ja 3 metriä syvä kuoppa. _Tavallista kiipeilyhakaa voidaan säilyttää turvallisesti sen selkämyksessä_.                                                                                                                                                                                       |
| **64**  | **Alkusyöksy**          | Olento kehittyy nopeasti lajinsa tulevaisuuden muunnokseksi. _Oudot näyt omista esivanhemmistaan kummittelevat omistajaa._                                                                                                                                                                                                 |
| **65**  | **Sysäys/Veto**         | Minkä tahansa kokoinen esine vedetään suoraan sinua kohti tai työnnetään suoraan sinusta poispäin yhden miehen voimalla. _Kirjaan kohdistuvat voimat voimistuvat koomisesti._                                                                                                                                              |
| **66**  | **Kuolleista nostatus** | Luuranko nousee maasta palvelemaan sinua. Ne ovat uskomattoman älyttömiä ja pystyvät tottelemaan vain yksinkertaisia käskyjä. _Omistaja kiehtoutuu luista jokaisen käyttökerran jälkeen yhä enemmän ja enemmän._                                                                                           |
| **67**  | **Hengen herätys**      | Lähellä olevan ruumiin henki ilmestyy ja vastaa yhteen kysymykseen. _Vastaukset (muttei niiden kysymykset) tulevat kirjoitetuksi ikuisesti sen sivuille._                                                                                                                                               |
| **68**  | **Ajatustenluku**       | Voit kuulla lähellä olevien olentojen pinnallisia ajatuksia. _Pitkäkestoinen käyttö voi saada lukijan pitämään toisten ajatuksia ominaan._                                                                                                                                                                                 |
| **69**  | **Hylkiminen**          | Kaksi esinettä hylkii toisiaan voimakkaan magneettisesti 3 metrin säteellä. _Suljettu kahdella voimakkaalla hihnalla, jotka aukeavat epäsopivina hetkinä._                                                                                                                                                                 |
| **70**  | **Kaukotarkkailu**      | Voit nähdä tänään aiemmin koskettamasi olennon silmien kautta. _Omistajan silmät muuttuvat maitomaisen valkoisiksi tunnin ajaksi käytön jälkeen._                                                                                                                                                                          |
| **71**  | **Aineen muovaus**      | Eloton materiaali käyttäytyy käsissäsi kuin savi. _Hitaasti hajoaa joutuessaan kosketuksiin puun tai kankaan kanssa. Hautaa multaan tai upota veteen palauttaaksesi kirja._                                                                                                                                |
| **72**  | **Aisti**               | Valitse jokin esine (avain, kulta, nuoli, kannu jne.). Voit aistia lähimmän vastaavan esineen. _Kirjan edellinen omistaja on aina tietoinen kirjan nykyisestä sijainnista._                                                                                             |
| **73**  | **Kilpi**               | Koskettamasi olento on suojattu tavanomaisilta hyökkäyksiltä minuutin ajan. _Sidottu ruosteiseen ristikkopanssariin ja on melko raskas. Jos hallussa, antaa +1 Panssaria._                                                                                                                                 |
| **74**  | **Verhous**             | Koskettamasi olento on näkymätön, kunnes se liikkuu. _Näkymätön muille paitsi kirjan nykyiselle omistajalle._                                                                                                                                                                                                              |
| **75**  | **Vaihdos**             | Kaksi näkemääsi olentoa vaihtaa välittömästi paikkaa keskenään. _Jos varastettu mutta ei vielä luettu, se ilmestyy takaisin sinne, minne sen omistaja sen viimeksi jätti._                                                                                                                                                 |
| **76**  | **Taitava korjaus**     | Teet pieniä korjauksia elottomaan esineeseen. _Sadan kirjan pergamentista ommeltu, yksikään sivu ei ole samanlainen._                                                                                                                                                                                                      |
| **77**  | **Nukutus**             | Näkemäsi olento vaipuu kevyeen uneen. _Pehmeä kuin tyyny, mutta antaa vain levotonta unta._                                                                                                                                                                                                                                |
| **78**  | **Liukkaus**            | Jokainen pinta 9 metrin säteellä muuttuu erittäin liukkaaksi. _Käsittelyyn tarvitaan käsineet, jottei kirja putoa mitä hullunkurisimmalla tavalla._                                                                                                                                                                        |
| **79**  | **Savumuoto**           | Kehosi muuttuu eläväksi savuksi, jota voit hallita. _Tuoksuu leirinuotiolta. Sivuja ei voi polttaa, mutta ne ovat hyvin herkkiä kosteudelle._                                                                                                                                                              |
| **80**  | **Nuusku**              | Voit haistaa hajujen hennoimmatkin jäljet. _Erittää voimakasta hajua, jonka vain sen omistaja voi havaita._                                                                                                                                                                                                                |
| **81**  | **Tukahdus**            | Minkä tahansa näkemäsi valon tavanomainen lähde sammuu välittömästi. _Jos jätetään pitkäksi aikaa samaan paikkaan, lähellä olevat valonlähteet alkavat himmenemään ja lopulta sammuvat._                                                                                                                                   |
| **82**  | **Järjestys**           | Elottomat esineet järjestävät itsensä asettamiesi ryhmien mukaan. _Oikaisee itsensä pystyyn, kun se pudotetaan tai heitetään._                                                                                                                                                                                             |
| **83**  | **Loitsusaha**          | Pyörivä terä lentää rinnastasi ja raivaa tieltään kaiken kasvillisuuden. Muuten se on vaaraton. _Kääritty tahriintuneeseen nahkaan, se on öljyttävä vähintään kerran kuukaudessa_.                                                                                                                         |
| **84**  | **Hämähäkkikiipeily**   | Pystyt kiipeilemään pintoja kuten hämähäkki. _Uudet hämähäkinseitit on työnnettävä syrjään ennen jokaista käyttökertaa. Niitä on vaikea poistaa._                                                                                                                                                          |
| **85**  | **Parvimuoto**          | Muutut parveksi variksia, rottia tai piraijoita. Sinua voidaan vahingoittaa vain _räjähdyshyökkäyksillä_. _Voidaan purkaa tusinaan osaan, jotka pikkuhiljaa alkavat liikkua takaisin toisiaan kohti._                                                                                                      |
| **86**  | **Houkutuskohde**       | Koskettamastasi esineestä tulee minkä tahansa lähellä olevan loitsun kohde. _Houkuttelee kaikenlaisia maagisia olentoja, loitsuvuotoja ja kaukotarkkailua._                                                                                                                                                                |
| **87**  | **Kaukoliikuttelu**     | Voit hengenvoimillasi siirtää yhden alle 30 kilon painoista esinettä. _Omistaja voi kutsua kirjan pelkällä henkisellä käskyllä (WIL-pelastusheitto tai kärsi _puutteesta_ sen jälkeen)._                                                                                                                |
| **88**  | **Telepatia**           | Kaksi olentoa voi kuulla toistensa ajatukset riippumatta siitä, kuinka kaukana toisistaan ne ovat. _Kantaja voi kuulla (mutta ei vastata) sen ajatuksiin, joka sen viimeksi omisti, ja päinvastoin._                                                                                                    |
| **89**  | **Teleportti**          | Näkemäsi esine tai henkilö siirtyy paikasta toiseen 15 metrin säteellä. _Voidaan tuhota portaalin luomiseksi toiseen ulottuvuuteen._                                                                                                                                                                                       |
| **90**  | **Tiheikkö**            | Puiden ja tiheän pensaikon muodostama tiheikkö, jonka leveys on enintään 15 metriä, puhkeaa yhtäkkiä esiin. _Kääritty köynnöksiin, jotka on hävitettävä aina uudelleen jokaisella käyttökerralla._                                                                                                                         |
| **91**  | **Ajanohjaus**          | Aika hidastuu tai nopeutuu 10% 30 sekunnin ajan 15 metrisen kuplan sisällä. _Ulkoasu vaihtelee mahdottoman vanhan ja painotuoreen välillä._                                                                                                                                                                                |
| **92**  | **Tosinäkö**            | Näet kaikkien lähellä olevien harhanäkyjen lävitse. _Ei voi peittää taikuudella ja erottuu helposti joukosta._                                                                                                                                                                                                             |
| **93**  | **Nousulähde**          | Merivesilähde ilmestyy. _Sidottu kovettuneisiin nahkakansiin, joihin on pinttynyt suolaa ja merirokkoja._                                                                                                                                                                                                                  |
| **94**  | **Ilmestys**            | Hallitset täysin sitä, minkä jokin olento näkee. _Etukantta koristaa häiritsevä luometon silmä._                                                                                                                                                                                                                           |
| **95**  | **Harhanäky**           | Valitsemasi kaltainen hiljainen, paikallaan oleva huoneen kokoinen harhanäky ilmestyy. _Täynnä rikkaan värikkäitä sivuja, jotka ovat aivan kuin lasten iltasadusta._                                                                                                                                                       |
| **96**  | **Vartiokuvio**         | Maahan ilmestyy hopeinen ympyrä, jonka läpimitta on 15 metriä. Valitse yksi laji, joka ei voi ylittää sitä. _Kansia koristavat omituiset ylimaalliset olennot, joilla on tuhansia silmiä._                                                                                                                 |
| **97**  | **Verkko**              | Kykenet ampumaan ranteistasi paksua verkkoa. _Teksti on vierasta, mutta jotenkin ymmärrettävää, sillä se on unien kieltä._                                                                                                                                                                                                 |
