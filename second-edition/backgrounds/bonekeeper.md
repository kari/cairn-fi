# Luunvartija

> Sinä olet poismenneiden paimen. Kuuntelet kuolleiden viimeisiä kuiskauksia, kun he laskeutuvat kylmään, taipumattomaan maahan. Tiedät että juhliaksemme täysin elämän lahjaa, meidän on kunnioitettava myös sen loppua.

## Nimiä

Rook, Ebon, Moro, Yew, Pall, Leth, Bea, Barnaby, Vesper, Leder

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Lyhty
- Öljykannu (6 käyttöä)
- Vaarna (d6)
- Ketjua (3 metriä)

## Mitä veit kuolleilta? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | **Variksen muotoinen amuletti**. Voit kysyä kuolleelta, mutta sinun on lisättävä **Väsymys** jokaisesta kysymyksestä. He eivät aina puhu totta.                                                                                                                                                                                                                                                                                                              |
| **2** | Vapautetun _kalmohengen_ antama **kuollettava haava**. Sinut parannettiin, mutta vääristymäsi on tehnyt sinusta hylkiön. Et tarvitse happea etkä ravintoa, mutta olet silti altis kivulle ja kuolemalle. Olet maailmojen välissä loukussa, ja kuolleet pitävät sinua yhtenä omistaan.                                                                                                                                                        |
| **3** | Paikallisen kuolemankultin **Verikiulu** (_kookas_). Kumoa se herättääksesi kuolleista palvelija, joka koostuu siitä mitä maan alle on haudattu ja jolla on 6 HP, 1 Panssari, 13 STR, 11 DEX, 4 WIL ja sirpalenyrkit (d8+d8). Vain yksi palvelija voidaan herättää kerrallaan. Jos se tuhoutuu, menetät pysyvästi 1d4 STR. **Lataus**: Täytä kuolevan soturin verellä. |
| **4** | **Ruumiskärryt** (+6 paikkaa, _hidas_) edellisestä työstäsi. Mukana tuli itsepäinen vanha **aasi** (+4 paikkaa, vain +2 paikkaa jos vetää vaunuja).                                                                                                                                                                                                                                                                                    |
| **5** | Muinaisesta kirjastosta varastettu **Taikuuden paljastus** -loitsukirja. Perheesi työskenteli hämäräperäisen tuonelan jumalan palveluksessa, mutta sinä menetit uskosi. Vaikka sinut onkin karkotettu luopiona, jatkat palvelusta. Taikuuden palajastus: Voit nähdä tai kuulla lähellä olevia maagisia auroja. _Tuntuu lämpimältä, kun taikuutta käytetään lähellä._                                         |
| **6** | **Ruttolääkärin naamio**, sen jälkeen kun sen omistaja menehtyi tautiin, joka hävitti kaikki tuttusi. Hänen olisi kannattanut pitää se naamallaan.                                                                                                                                                                                                                                                                                                                           |

## Mikä työkalu oli korvaamaton työssäsi? Heitä 1d6:

|   |                 |                                                                                                                                                                                                                        |
| - | --------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1 | **Kahleet**     | Vaikkakin vanhat, edelleen tehokkaat jopa hyvin vahvoja vastaan. Sinulla ei ole avainta.                                                                                               |
| 2 | **Pesusieni**   | Kuulemma valmistettu harvinaisen merenelävän jäännöksistä. Se ei tunnu koskaan kuivuvan.                                                                                               |
| 3 | **Väkipyörä**   | Sopii erinomaisesti hautakivien, kivien tai jopa ruumiiden siirtämiseen.                                                                                                                               |
| 4 | **Suitsuke**    | Täydellinen rituaaleihin tai pitämään kärpäset loitolla. Rauhoittaa verta.                                                                                                             |
| 5 | **Sorkkarauta** | d6 vahinkoa. Joskus on vain saatava se hemmetin juttu auki!                                                                                                                                            |
| 6 | **Karkote**     | Voimakasta tavaraa. Sen haalistunut etiketti tekee kuitenkin epäselväksi, mitä sen oikeastaan on _tarkoitus_ on karkottaa. Kenties kaikkea. 3 käyttöä. |
