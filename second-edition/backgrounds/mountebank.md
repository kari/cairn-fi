# Puoskari

> Nokkeluus on terävin aseesi, kulissisi vahvin kilpesi. Mutta kun häviät, häviät pahasti.

## Nimiä

Ambrose, Lucius, Beauregard, Cornelius, Aria, Toph, Indigo, Delphine, Solene, Noa

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Kärryt (+4 paikkaa, _kookkaat_ vedettäessä)
- Huijarin pelikortit
- Tyylikäs hattu (_mitätön_)
- Kävelykeppimiekka (d6)

## Miten petoksesi paljastui? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                                                                |
| ----- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | "Potilaasi" raportoivat jatkuvasti ihmetoipumisista, vaikka sinulla ei ole minkäänlaista koulutusta. Aloita **Sidetarpeilla** (3 käyttöä) ja taidolla parantaa.                                                                                                                                                                                                                                                |
| **2** | Sen jälkeen kun olit vietellyt varakkaan suojelijan, hänen perheensä palkkasi rikollisjengin hakemaan sinut. Pääsit karkuun ja nyt sinun on pysyttävä piilossa. Aloita **Kauneusvoiteen** kanssa, 2 käyttöä. Levitä näyttääksesi vastustamattoman kauniilta seuraavien 12 tunnin ajan.                                                                                                                         |
| **3** | Kauppasit keksittyjä ennustuksia, mutta kun yksi niistä kävikin toteen, se herätti ei-toivottua huomiota. Heitä **Enne**-taulukkoa, mutta pidä tulos itselläsi. Aloita piilotettavalla **Veitsellä** (d6, _mitätön_).                                                                                                                                                                                          |
| **4** | Viimeisin temppusi tuhosi korvaamattoman esineen ja kymmenkunta sivullista loukkaantui. Aloita **Kapteenin univormulla** (_mitätön_), **Seremoniallisella miekalla** (vaaraton, 60gp) ja **Kukkakimpulla**.                                                                                                                                                                                                    |
| **5** | Puskanoita kirosi sinut, koska huijasit viattomia kyläläisiä. Taikuus toimii käsissäsi arvaamattomasti (_WIL-pelastusheitto katastrofin välttämiseksi_). Jos olet taikuuden kohteena, sama koskee myös sen käyttäjää.                                                                                                                                                                                          |
| **6** | "Spiritismi-istuntosi" kuolleiden kanssa olivat todellisuudessa huijauksia, johon liittyi taitavasti piilotettu Harhakuulo -loitsukirja. Väistämättä joku suojelijoistasi sai lopulta selville salaisuutesi. Aloita **Loitsukirjalla** ja nipulla huiveja. Harhakuulo: Luot harhaääniä, jotka vaikuttavat tulevan valitsemastasi suunnasta. Tuottaa satunnaisia ja toisinaan sopimattomia ääniä päivän aikana. |

## Mistä muistoesineestä sinut voi aina tunnistaa? Heitä 1d6:

|       |                            |                                                                                                                                                                                          |
| ----- | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Kuninkaallinen vaakuna** | Synnyit kuninkaalliseen perheeseen, mutta valitsit toisenlaisen elämän. Vaakunan avulla pääset paikkoihin, mutta se myös paljastaa olinpaikkasi perheellesi. _Mitätön_.                  |
| **2** | **Ihmeöljy**               | Haiseva, liukas seos. 2 käyttöä.                                                                                                                                                         |
| **3** | **Kirurgin saippua**       | Lipeästä ja tuhkasta valmistettu palasaippua, joka tekee ihon väliaikaisesti läpinäkyväksi ja paljastaa anatomian sen alla. 4 käyttöä.                                                   |
| **4** | **Vuohenjauhe**            | Peräisin vuohenpoikasen istukasta.  Parantaa tilapäisesti minkä tahansa vaivan, mutta oireet palaavat muutamassa tunnissa.                                                               |
| **5** | **Kirottu safiiri**        | Arvoltaan **200gp**, se palaa näkyvästi pian sillä maksamisen jälkeen takaisin taskuusi. Et näytä pääsevän siitä eroon.                                                                  |
| **6** | **Alkemistinen Tatuointi** | Koira, kissa tai lintu, joka voi irtautua kehostasi halutessasi. Se noudattaa käskyjäsi parhaan kykynsä mukaan ja voi välittää vammansa (_STR-menetyksenä_) takaisin sinulle. _Mitätön_. |
