# Välskäri

> Kuljet parantajan ja rääkkääjän välimaastossa, tunnet niin lihan haurauden kuin sen piilevät salaisuudet. Oikeilla työkaluilla elämä ja kuolema ovat vain sanoja.

## Nimiä

Wilmot, Patch, Lancet, Sawbones, Theo, Cutwell, Humor, Landsford, Goodeye, Johanna

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Luusaha (d6)
- Sidetarpeet (3 käyttöä)
- Verenimijä (palauttaa 1 STR, 3 käyttöä)
- Tahriintunet hoitovaatteet (_mitätön_)

## Miten olet "parantanut" itseäsi? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                   |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | Sinulla on vaihto**silmä**, joka voi suurentaa esineitä, toimia kaukoputkena ja antaa minimaalisen pimeännäön. Et voi pitää päässäsi mitään metallista, ja voimakkaiden magneettien läsnäolo aiheuttaa sinussa _puutostilan_.                                                     |
| **2** | Yksi **jalka** on enimmäkseen metallia (potku, d6), ja kohtelet joitakin Vaikeita maastoja **Helppoina**. Kanna **Öljykanisteria** (6 käyttöä). Ilman päivittäistä öljyämistä kärsit _puutteesta_ ja olet meluisa.                                                                |
| **3** | Yksi sormistasi on vaihdettu, sen luut on korvattu kullalla ja raudalla. Ota **koukku** ja **ruuvimeisseli**, jotka voidaan kiinnittää sormenpäähän.                                                                                                                              |
| **4** | Molemmat **korvat** on kirurgisesti paranneltu, mikä kolminkertaistaa kuulosi. Voit keskittyä tiettyyn ääneen, kuten keskusteluun, suuren etäisyyden yli. Pidät korvaläppää, joka suojaa äkillisiltä kovilta ääniltä (WIL-pelastusheitto välttääksesi tilapäisen halvaantumisen). |
| **5** | **Rintakehäsi** on peitetty alkemistien kuvioilla, jotka _vahvistavat ihosi_ (1 Panssari). Muiden metallihaarniskojen käyttäminen mitätöi vaikutuksen.                                                                                                                            |
| **6** | Toinen **käsi** on täysin metallinen ja irtoaa olkapäästä. Sitä voidaan käyttää aseena (d8, _kookas_ kun se ei ole kiinnitetty) ja se voi liikkua itsenäisesti, jos olet sen näköetäisyydellä.                                                                                    |

## Mikä harvinainen työkalu on välttämätön työssäsi? Heitä 1d6:

|       |                           |                                                                                                                                                                                                                                     |
| ----- | ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Uudelleenkasvun voide** | Kasvattaa ruumiinosan takaisin yhden vuorokauden aikana. 1 käyttö.                                                                                                                                                                  |
| **2** | **Siirrännäistoukka**     | Pieni mato, joka voi sulattaa elottomia esineitä kiinni ruumiinosiin.  1 käyttö.                                                                                                                                                    |
| **3** | **Haavavaha**             | Parantaa tulen tai kemikaalien aiheuttamat haavat (palauttaa täyden STR:n), mutta ei mitään muuta. 2 käyttöä.                                                                                                                       |
| **4** | **Elohopea**              | Piriste. Mene ensimmäisenä taistelussa ja läpäise automaattisesti kaikki WIL-pelastusheitot yhden tunnin ajan. _Riippuvuus_: Tee STR-pelastusheitto tai joudu _puutostilaan_ 24 tunnin kuluttua ilman sitä. 4 käyttöä.              |
| **5** | **Keuhkopumppu**          | Siirrettävä rautakeuhko (_kookas_). Mahdollistaa hengenpelastavan leikkauksen tai vedenalaisen hengityksen.                                                                                                                         |
| **6** | **Kulkukivi**             | Vetää vaaralliset aineet pois kehosta ja toimii voimakkaana magneettina.                                                                                                                                                            |
