# Taskuvaras

> Elät harmaalla alueella niiden, joilla on valtaa, ja niiden, joilla ei ole, välissä. Löydät tilaisuuksia sieltä missä muut näkevät vain kaaosta. Vikkelillä sormilla kevennät niin rikkaimman kauppiaan että alhaisimman vartijan taskuja.

## Nimiä

Arlo, Lyra, Eamon, Salina, Elara, Freya, Bull, Sparrow, Ivy, Silas

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Kaksoistikarit (d6+d6, _kookas_)
- Topatut nahkavaatteet (1 Panssari)
- Tiirikka
- Musta asu (_mitätön_)

## Mikä oli viimeinen iso keikkasi? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                         |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | Aatelisen kesäkoti. Paikka oli täynnä hienoja viinejä (+20gp), muttei paljon muuta. Ota **Aitaleikkurit**.                                                                                                           |
| **2** | Pankki. (Jäit kiinni.) Sinulla on polttomerkki, joka näkyy vain tulen valossa, ja jokainen sen nähnyt saa pyytää sinulta oluen. Ota **Sisäänvedettävä vaijeri**.                                     |
| **3** | Killan varasto. Ota **Tikkaat** (_kookkaat_, 3 metriä) ja **Häikäisyjauhe** (1 käyttö).                                                                                                                           |
| **4** | Rahanlainaaja. Joku ehti ennen sinua, mutta jätti jälkeensä _Loihtosilmän_ **Käärön** (_mitätön_). _Loihtosilmä_: Voit nähdä läpi maagisen leijuvan silmämunan, joka lentelee ympäriinsä käskystäsi. |
| **5** | Suurkonstaapelin asunto. Pakenit, mutta jätit ystäviä jälkeesi. Ota **Vahva silkkiköysi** (9 metriä) ja levoton olo.                                                                                                 |
| **6** | Yliopisto. Sinut nähtiin, mutta sinua ei jahdattu. Et vieläkään tiedä miksi ei. Ota **Savukiekot** (3 käyttöä).                                                                                      |

## Mikä auttaa sinua varastamisessa? Heitä 1d6:

|       |                    |                                                                                                                                                                                                                                                       |
| ----- | ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Katinsormus**    | 2 latausta. Kiipeä seinä ylös ja tipu alas turvallisesti. **Lataus**: Aseta sormus kulkukissan häntään.                                                                               |
| **2** | **Kultasormi**     | 1 lataus. Sormikas, joka pystyy jäljittelemään mitä tahansa tavallista avainta. **Lataus**: Täytä se vähintään 100 kultarahalla yön yli.                                              |
| **3** | **Kurkistuslasi**  | 3 käyttöä. Monokeli, jonka avulla näet seinien tai muiden esteiden läpi. Se särkyy viimeisen käyttökerran jälkeen sirpaleiksi.                                                                        |
| **4** | **Sulopilli**      | 1 lataus. Kuulijat kuulevat kaukaa pehmeän, tutunoloisen äänen, jota he eivät voi olla seuraamatta. **Lataus**: Menetä rakas muisto. (Kuvaile se.) |
| **5** | **Kulkurin huivi** | 1 lataus. Käytä sitä sulautuaksesi väkijoukkoon, esiintyen tavallisena kerjäläisenä. **Lataus**: Lahjoita päivän voitot köyhille. _Mitätön_                                           |
| **6** | **Käänteishyrrä**  | 1 käyttö. Pyöräytettäessä aika hyppää 30 sekuntia taaksepäin. Kaikki muistavat mitä tapahtui.                                                                                                         |
