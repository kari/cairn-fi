# Noidanpuolikas

> Syntynyt sekä kuolevaisten että piilotetusta maailmasta, olet joillekin arvoitus ja monille pelätty. Sinun on tarina siitä kun kaksi maailmaa törmää toisiinsa.

## Nimiä

Solena, Veles, Bryn, Sabine, Razvan, Rowena, Galen, Nyx, Vex, Iwan

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Loitsukirja (Tiheikkö: Puiden ja tiheän pensaikon muodostama tiheikkö, jonka leveys on enintään 15 metriä, puhkeaa yhtäkkiä esiin. _Kääritty köynnöksiin, jotka on hävitettävä aina uudelleen jokaisella käyttökerralla._)
- Rautatikari (d6)
- Yrttipussi (palauttaa 1 STR, 3 käyttöä)
- Ghilliepuku

## Mitä toit mukanasi Hämärähovista? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                             |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Mustaruusuviulu** (_kookas_). Sen musiikki aiheuttaa läheisissä kuolevaisissa voimakasta surua ja kykenemättömyyttä liikkua. (Muut ovat vain kiehtoutuneita.) Et osaa soittaa sitä.                                                                                 |
| **2** | **Paperijalat**. Olet äärimmäisen kevyt, ja voit tippua muutaman kerroksen verran ilman että loukkaannut. Yritä välttää niiden repeytymistä tai kastumista.                                                                                                                                                                 |
| **3** | Sisälläsi asustaa **Elävä painajainen**, mutta se ilmenee aina kun olet vaarassa. Sillä on kanssasi samat **Kyvyt** ja **HP** ja se hyökkää kynsillään (d8+d8). Se katoaa **Kriittisen vahingon** yhteydessä (ota 1d4 WIL-vahinkoa), ilmestyen uudelleen seuraavan täydenkuun aikaan. |
| **4** | **Korppiapulainen** [8 HP, 3 STR, 11 DEX, 13 WIL, nokka, (d6)]. Se puhuu kuin älykäs olento ja on täysin omistautunut sinulle.                                                                                                                                       |
| **5** | **Pensasaidan piikki**. Se voi lävistää minkä tahansa orgaanisen materiaalin (melko kivuliaasti), mutta kun se vedetään irti, se ei jätä jälkeäkään.                                                                                                                                                                     |
| **6** | Keijuolennon **Tosinimi**. Lausu se kutsuaksesi sen omistaja tekemään suuri palvelus, mutta vain kerran. Oikealta ostajalta siitä voi myös saada muhkean hinnan.                                                                                                                                                            |

## Mitä keitosta sinulla on mukanasi, ja mitä harvinaisia aineksia keräsit sen valmistamiseksi? Heitä 1d6:

|       |                               |                                                                                                                                                                                                                                            |
| ----- | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | **Uudelleensyntyminen tuhka** | Kaarnahengen jäänteet. Ripottele sytyttääksesi uudelleen sammunut tuli tai herättääksesi henkiin vain hetkeä aiemmin kuolleen olennon. 3 käyttöä.                                          |
| **2** | **Hohtosulka**                | Tulilinnun höyhen. Voi saada minkä tahansa olennon esiintymään vakuuttavasti jonain muuna olentona. 1 käyttö.                                                                              |
| **3** | **Orapihlajan siemen**        | Terho toiselta puolelta, kerätty kevätpäiväntasauksen aikaan. Kun se istutetaan, se versoo ylellisen suojan, joka romahtaa kuun noustessa seuraavana päivänä. 1 käyttö.                    |
| **4** | **Kivipuun mahlaa**           | Verta vastaan saatu mahla. Kovettuu, kun sitä hierotaan mihin tahansa pintaan (+1 Panssari). 3 käyttöä.                                                                 |
| **5** | **Yönpölyn jauhe**            | Valmistettu kuuden pöllön rituaalipoltosta. Kun sitä heitetään ilmaan, päivä muuttuu hetkeksi yöksi. 2 käyttöä.                                                                            |
| **6** | **Kirouskivi**                | Kerätty joesta, joka virtaa _toiselta puolelta_. Rautapurkistaan poistettuna se voi imeä itseensä aktiivisen taian vaikutukset. Jos se tuhotaan, taika vapautuu. 1 käyttö. |
