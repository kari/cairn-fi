# Löytölapsi

> Outo syntymämerkki, kummallinen haju: jotenkin _muun_ kosketus viipyy yhä. Missä tahansa oletkin, sinulla on vaikeuksia sopeutua joukkoon. Heitä **Enne**-taulukkoa (vaikka et olisikaan nuorin pelaaja), mutta pidä tulokset toistaiseksi omana tietonasi.

## Nimiä

Faunus, Snowdrop, Wisp, Silverdew, Brim, Solstice, Steeleye, Artea, Gossamer, Hazel

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Suolapussukka
- Perintökoru (_mitätön_, hehkuu taikuuden läsnä ollessa)
- Linko (d6)
- Tikari (d6)

## Kuka otti sinut huomaansa? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                 |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | Vanha metsästäjä. Olitte molemmat varsin onnellisia, kunnes se kaikki päättyi. Ota **Kulunut pitkäjousi** (d8, _kookas_) ja **Nahkainen Jerkin-liivi** (1 Panssari).                                                                                                                                                                                            |
| **2** | Kurttuinen apteekkari, joka opetti sinulle parantajan taitoja, mutta säilytti kliinisen etäisyyden sinuun. Ota **Parantava voide** (palauttaa d4 STR, 1 käyttö).                                                                                                                                                                                                |
| **3** | Druidi, joka opetti sinulle puiden kielen. Kun tuli aika, otit mukaasi vain **Kieron sauvan** (d8) ja jätit lupauksen, että palaisit jonain päivänä.                                                                                                                                                                                                            |
| **4** | Äreä seppä uneliaasta jokikaupungista. Sinuun pidettiin aina etäisyyttä. Nyt ahjo on kylmä ja sinä olet siirtynyt eteenpäin. Ota **Sepän esiliina** (_mitätön_) ja **Useasti korjattu rengaspanssari** (2 Panssaria, _kookas_).                                                                                                                                 |
| **5** | Kiertävien viihdyttäjien joukko. Jonkin aikaa he olivat sinulle kuin perhettä. Eräänä päivänä herätessäsi he olivat lähteneet ilman mitään selitystä. Ota **Satukirja**, **Tikari** (d6) ja joitakin polttavia kysymyksiä.                                                                                                                                      |
| **6** | Syrjäisen metsäluostarin munkit. Kun heidän sääntönsä alkoivat tuntua liian tiukoilta, sinä hiivit pois. Ota **Munkin kaapu** (lämmin, _mitätön_) ja _Kasvien Hallinnan_ **Loitsukirja**. Kasvien Hallinta: Lähistöllä olevat kasvit ja puut tottelevat sinua ja saavat kyvyn liikkua hitaasti. _Lehtiä kasvaa pitkin selkääsi ja tuoksu on hieman mädäntynyt._ |

## Mikä pitää ikävät uutiset loitolla? Heitä 1d6:

|       |                    |                                                                                                                                                                             |
| ----- | ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Piipputupakka**  | Onnenkalusi. Keskustelut sujuvat yleensä helpommin tupakan jälkeen. 6 käyttöä.                                                                                              |
| **2** | **Haisupurkki**    | Tämän purkin rikkominen vapauttaa niin pahan hajun, että kaikkien lähistöllä olevien on tehtävä STR-pelastusheitto tai oksennettava välittömästi. 1 käyttö.                 |
| **3** | **Murattimato**    | Vihreä mato, jota usein luullaan rikkaruohoksi. Nielaistuna kokonaisena se imee itseensä kaikki kehossa olevat myrkyt tai mädät ennen poistumistaan tavanomaisella tavalla. |
| **4** | **Unikivi**        | Sileä sininen kivi, joka auttaa muistamaan unet selkeämmin. Liiallinen käyttö voi aiheuttaa uniriippuvuutta.                                                                |
| **5** | **Sukellusvarpu**  | Sormenmittainen puukeppi, jonka koko kaksinkertaistuu aina kun se upotetaan kokonaan veteen. Se ei kutistu uudelleen.                                                       |
| **6** | **Jäniksenkäpälä** | Sinulla oli se, kun sinut löydettiin. Sanotaan, että se on sinut jättäneen naisen jalka, ja että se suojaa sinua noitien taikuudelta. _Mitätön_.                            |
