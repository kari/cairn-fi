# Jonglööri

> Mikä inspiroi sielua enemmän kuin laulut, sanat ja näytelmät? Miksi harjoitella vuosikausia hallitsemaan salaperäisyyksiä, kun sisälläsi on jo todellista taikuutta?

## Nimiä

Jax, Selene, Baladria, Ada, Mort, Saylor, Tripp, Lantos, Echo, Jubilo

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Asu
- Yksinkertainen soitin (huilu, luuttu jne.)
- Onnekas jerkin-liivi (1 Panssari)
- Linko (d6)

## Mitä tapahtui viimeisessä esityksessäsi? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ----- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | Huolimatta tappavien taitojen koulutuksestasi, näyttelijä kuoli ja sinä sait syytökset. Ota kevyt **Säilä** (d6) ja väärä henkilöllisyys.                                                                                                                                                                                                                                                                     |
| **2** | Yleisö piti tarttuvasta kappaleestasi, joka kertoi aatelisesta ja hänen romanttisista epäonnistumisistaan. Kyseinen aatelinen, ei niinkään. Ota _Ajatustenluku_-**loitsukirja** ja pidätysmääräyksesi. Ajatustenluku: Voit kuulla lähellä olevien olentojen pinnallisia ajatuksia. _Pitkäkestoinen käyttö voi saada lukijan pitämään toisten ajatuksia ominaan._ |
| **3** | Debyyttisävellyksesi ajoi yleisösi niin sekaisin, että he mutisivat hölynpölyä yötaivaalta laskeutuvista kirkkaista olennoista. Myöhemmin huomasit, että nuotit muistuttivat tähtikuvioita. Ota **Kirja tähtitieteestä** ja joukko kysymyksiä.                                                                                                                                                                   |
| **4** | Pilkkasit unohdettua huijarijumalaa ja sinut kirottiin siitä hyvästä. Kykenet puhumaan vain riimitellen. Ironista kyllä, tämä on tehnyt sinusta vain suositumman vertaistesi kanssa. Ota **Synonyymisanakirja** (20gp). Ilman sitä olet _puutteessa_.                                                                                                         |
| **5** | Sait arven lavalla sattuneessa onnettomuudessa. Yleisö hurrasi, koska luuli sen kuuluvan esitykseen. Ota hyvin kulunut **Lavahaarniska** (1 Panssari), mieleenpainuva arpi ja aplodien pelko.                                                                                                                                                                                                 |
| **6** | Kunnioitettavaa nukketeatteritaitoasi vastasi vain matkimiskykysi. Olit niin hyvä, että sinut leimattiin noidaksi (kirjaimellisesti) ja karkotettiin. Ota **Epätodellinen käsinukke** ja **Kanin pääkallo** (_mitätön_), joka suojaa loitsuilta.                                                                                                                           |

## Mitä rihkamaa et pystynyt jättämään jälkeesi? Heitä 1d6:

|       |                             |                                                                                                                                                            |
| ----- | --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Väärät käsiraudat**       | Mukavat, aidon näköiset käsiraudat. Vain sinä tiedät tempun, jolla niistä pääsee irti.                                     |
| **2** | **Taskuteatteri**           | Sarja pieniä nukkeja ja kokoontaitettava näyttämö. Hyvä nopeaan harhautukseen.                                             |
| **3** | **Aaveviulu**               | Tummanharmaa viulu, joka soittaa vaivaavaa sävelmää, jota toistaa näkymätön, kaukainen kaksonen.                                           |
| **4** | **Traagiset tarinat**       | Asiallisessa seurassa kielletty kirja muuttuu loppua kohden vähemmän riettaaksi ja vain ahdistavammaksi. Arvo **100gp**.   |
| **5** | **Myyttinaamio**            | Kipsinaamio, jonka avulla voi ottaa hirviön kasvot ja ilmeet. Kun se otetaan pois, lisää **Väsymys**.                      |
| **6** | **Uudelleenrikottava lasi** | Viinilasi, joka voidaan rikkoa useita kertoja ja joka palaa takaisin kasaan 24 tunnin kuluttua. Pitää todella kovaa ääntä. |
