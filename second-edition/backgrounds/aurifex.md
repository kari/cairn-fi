# Salaseppä

> Olet loihtimisen käsityöläinen, hienovaraisten voimien seppä. Työpajasi sulatusuunissa tätä maailmaa säätelevät lait taittuvat sinun tarpeittesi mukaan.

## Nimiä

Hestia, Basil, Rune, Prism, Ember, Quintess, Aludel, Mordant, Salaman, Jazia

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Lyhty
- Öljykannu (6 käyttöä)
- Neulaveitsi (d6)
- Suojakäsineet (_pikkuiset_)

## Mikä meni kauhistuttavan pieleen? Heitä 1d6:

|       |                                                                                                                                                                                                                                                  |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | Tapahtui räjähdys ja menetit hajuaistisi. No, melkein: voit nuuskia kultaa kuin sika tryffeleitä. Ota **Nuuskapurkki** (6 käyttöä) vaimentamaan vaikutusta. Käytä sitä joka päivä tai kärsit _puutteesta_.                                       |
| **2** | Teit rakkaan lemmikin aineettomaksi. Nyt se seuraa sinua ympäriinsä, näkymättömänä mutta aina läsnä. Vaikka se ei voi olla vaikuttaa fyysiseen maailmaan, pystyt jakamaan sen aistit. (Lisää **Väsymys** joka kerralla.) Se seuraa peruskäskyjä. |
| **3** | Altistuit pitkävaikutteiselle totuusseerumille, jonka vaikutus ei ole vielä ehtinyt loppua. Häiriöllä on etunsa: et voi myöskään toistaa kuulemiasi valheita.                                                                                    |
| **4** | Olit taitava luomaan _väärennettyä_ kultaa, joka oli lähes kuin oikeaa. Lopulta huijauksesi paljastui, ja sinun piti pikaisesti paeta. Ota painava **Metalliharkko** ja **Kultapölyä** (3 käyttöä).                                              |
| **5** | Alkemiakaavasi toimi, mutta kilpailijasi varasti luonnoksesi ennen kuin väitteesi voitiin todistaa oikeaksi. Ota kokeellinen **Väkipyssy** (d12, _räjähdys_, _kookas_), jonka lataaminen kestää yhden kierroksen, sekä kostonhimo.               |
| **6** | Sinua pilkattiin siitä, että keksit miten kulta muutetaan _lyijyksi_, ja jouduit naurunalaiseksi. Ota pullo **Yleisliuotinta** (2 käyttöä), joka liuottaa kaiken mihin se koskettaa ainesosiinsa.                                                |

## Mikä alkemian ihme on nerokkuutesi viimeisin tuote? Heitä 1d6:

|       |                             |                                                                                                                                                                                                                                                                                               |
| ----- | --------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Itsestään syttyvä geeli** | Tahmea vihreä neste, joka syttyy tuleen, kun se joutuu ilman kanssa kosketuksiin ja palaa sitten 8 tuntia. Ei voida sammuttaa (1 käyttö).                                                                                                     |
| **2** | **Räjähdyspallo**           | Räjähdysainejauheella täytetty pään kokoinen rautapallo, joka räjähtää osuessaan (d12, _räjähdys_, _kookas_, 1 käyttö).                                                                                                                       |
| **3** | **Akvaviitti**              | Puhdistaa minkä tahansa nesteen, muuttaen sen puhtaaksi vedeksi. Sen juominen parantaa 1d6 STR (1 käyttö).                                                                                                                                    |
| **4** | **Matkimiskivi**            | Tallentaa lyhyen lauseen, joka voidaan myöhemmin toistaa uudelleen.                                                                                                                                                                           |
| **5** | **Kipinäpöly**              | Syttyy helposti ja nopeasti. Hyödyllinen sytykkeenä tai palopommina (3 käyttöä).                                                                                                                                                              |
| **6** | **Homunculus**              | Pienikokoinen savikopio itsestäsi, joka noudattaa jokaista käskyäsi. Se inhoaa olla sinun vallassasi ja valittaa katkerasti aina kun mahdollista. Kaikki homunculukselle aiheutunut vahinko aiheutuu myös sinulle. 3 HP, 4 STR, 13 DEX, 5 WIL |
