# Sienestäjä

> Seuraat kuiskauksia syvältä maan alta, pinnan alla kasvavan sienirihmastometsän rytmikästä sykettä. Pimeys ei kauhistuta sinua. Lisäksi sinä todella rakastat sieniä.

## Nimiä

Unther, Woozy, Hilda, Current, Leif, Ratan, Mourella, Lal, Per, Madrigal

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Teroitettu istutuslapio (d6)
- Kynttiläkypärä (+1 Panssari, himmeä, 6 käyttöä)
- Köysi (25 jalkaa)
- Metalliämpäri

## Minkä oudon sienen löysit? Heitä 1d6:

|       |                                                                                                                                                                                           |
| ----- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Kirkuva torvisieni**. Kun se altistuu valolle, se huutaa niin kovaa, että kaikki lähistöllä olevat hyökkäykset (myös omasi) _heikentyvät_. 2 käyttöä.                                   |
| **2** | **Soihtusieni**. Murskattuna luo lyhyen aikaa kylmää sinistä valoa. 2 käyttöä.                                                                                                            |
| **3** | **Murhanhimoinen tryffeli**. Pistävän hajuinen, erittäin myrkyllinen ja hyvin harvinainen (50gp:n arvoinen salamurhaajille).  Laitonta melkeinpä kaikkialla. 1 käyttö.                    |
| **4** | **Hornanlakki**. Sen tuoksulle altistuminen aiheuttaa voimakasta pahoinvointia ja oksentelua. Joka tapauksessa se tyhjentää tilan. Pullotettu (1 käyttö).                                 |
| **5** | **Versomaljakas**. Nauti kutistuaksesi hiiren kokoiseksi. (Omaisuutesi pysyy samankokoisina.) Palaat normaalikokoon tunnin kuluessa, usein epätasaisesti. 1 käyttö.                       |
| **6** | **Juurikukka**. Valkoinen sieni, jota löytyy vain syvällä maan alla olevista ruumiista. Nauti palauttaaksesi d6 WIL-pistettä. Näet unta kuolleista ja heidän tarinoistaan. 1 käyttö.      |
|       |                                                                                                                                                                                           |

## Mikä pitää sinut järjissäsi, jopa täydellisessä pimeydessä? Heitä 1d6:

|       |                            |                                                                                                              |
| ----- | -------------------------- | ------------------------------------------------------------------------------------------------------------ |
| **1** | **Hehkuetana**             | Levittää pehmeää bioluminesenssivaloa. Syö yhden muona-annoksen joka toinen päivä.                           |
| **2** | **Silkkiperhon huivi**     | Säänkestävä peite, jolla voi myös sammuttaa tulipalon sen vahingoittumatta.                                  |
| **3** | **Valkokukka**             | Hienovarainen piriste. Sen pureskelu tekee sinusta immuunin _paniikille_ seuraavan tunnin ajaksi. 3 käyttöä. |
| **4** | **Päivänpaisteen osoitin** | Humisee hiljaa lähestyessään kohti Aurinkoa. Lopulta melu muuttuu sietämättömän kovaksi.                     |
| **5** | **Laiskiaissuojus**        | Luja ja säänkestävä kangas, käytännöllinen puuhun ripustettuna. Sen sisällä sinulla on +1 Panssari.          |
| **6** | **Kaivosrasva**            | Irrottaa jalokiven, työkalun tai raajan tiukasta raosta. Erittäin räjähdysherkkää. 3 käyttöä.                |
