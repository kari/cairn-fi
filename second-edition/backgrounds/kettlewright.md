# Kattilamaakari

> Sinut tunnetaan sulan metallin hajusta ja tinan helinästä. Et ole pelkkä kauppias vaan tulen ja metallin käsityöläinen.

## Nimiä

Fergus, Eon, Bram, Idris, Hester, Darragh, Seren, Rónán, Berek, Lorenz

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Pihdit
- Tinarulla
- Käsineet (_mitätön_)
- Vasara (d6)

## Mikä on alaasi? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                    |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | Rakennat pieniä härveleitä paikallisille killoille (etkä kysele liikaa). Ota 40gp ylimääräistä sekä etsintäkuulutusjuliste naamallasi. Jos aikaa ja materiaaleja on riittävästi, voit avata melkein minkä tahansa oven tai holvin.              |
| **2** | Kotitarvikkeet ja työkalut ovat omaa alaasi, ja kaupittelet tuotteitasi kaupunkilaisille eri puolilla maata. Ota 20gp:n arvosta esineitä [varustetaulukosta](/second-edition/players-guide/marketplace#gear). Puhut sujuvasti Matkaajien Murretta. |
| **3** | Olit sotavoimissa sulattaja, ennen kuin rauha tuhosi elinkeinosi. Ota **sulatusvasara** (d10, _kookas_) ja **tinakypärä** (+1 Panssari). Jos aikaa ja materiaaleja on riittävästi, voit korjata haarniskoita.                |
| **4** | Myyt harvinaisia ja laadukkaita esineitä niin luostareille kuin aatelisillekin. Ota **Kaukoputki**, **Kaulakoru** (_mitätön_, arvo 20gp), ja [Peilivaelluksen](/second-edition/players-guide/spellbooks) **Käärö** (_mitätön_).              |
| **5** | Tarjoat suojelupalvelua, vahdit hiljaa uhkien varalta kun raha vaihtaa omistajaa. Sinulla on aluksi **+d4 HP**:ta ja kannat **Pitkämiekkaa** (d10, _kookas_) ja **Gambesonia** (+1 Panssari).                                |
| **6** | Keräät raakaa tinaa ja rautaa taistelukentiltä, ja vedät hampaita vielä kouristelevilta ruumiilta. Aloita nuorella **Aasilla** (+4 paikkaa, hidas), **Varsijousella** (d8 vahinkoa, _kookas_) ja **Sahalla** (d6).        |

## Mikä ei koskaan petä sinua pääsemään vaikeuksista? Heitä 1d6:

|   |                      |                                                                                                                                                                                                                                                                        |
| - | -------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1 | **Tulimunat**        | Kuusi pientä merisuolasta, puusta ja savipölystä valmistettua puristettu pallo. Ne räjähtävät alhaisessa lämmössä (d8, _räjähdys_), mutta liekit haihtuvat nopeasti.                                                |
| 2 | **Musta terva**      | Monipuolista: Sekä tahmeaa että helposti syttyvää. 3 käyttöä.                                                                                                                                                          |
| 3 | **Piikkisaappat**    | Murtaa päitä (d8) yhtä helposti kuin jäätä ja sontaa. Matkustaminen on myös hieman hitaampaa, mutta helpompaa.                                                                                                      |
| 4 | **Nikkarointiliima** | Sulkee tiiviisti minkä tahansa nyrkin kokoisen aukon. 3 käyttöä.                                                                                                                                                                       |
| 5 | **Ilotulitteet**     | Häikäisevä vaikkakin vaarallinen esitys. Tarpeeksi räjähdysainetta pamauttaa irti sormi tai pari. 2 käyttöä jäljellä.                                                                                                  |
| 6 | **Haaskakissa**      | Nokkela lemmikki, joka on tarpeeksi pieni lymyilläkseen repussasi (_kookas_), mutta tarpeeksi voimakas pitääkseen pienemmät saaliseläimet poissa. Vaatii yhden **muona-annoksen** päivässä, ja sen on oltava lihaa. |
|   |                      |                                                                                                                                                                                                                                                                        |
