# Pellonvartija

> Sadon suojelijoita, jotka puolustavat sitä tuholaisia, varkaita ja petoja vastaan. Suuri kunniatehtävä, niin kauan kuin sitä kestää: monet vartijat eivät elä luonnollista elämäänsä loppuun asti. Heitä toisen kerran **Siteet**-taulukolla.

## Nimiä

Seed, Thresh, Dibber, Sow, Stalk, Harrow, Cobb, Flax, Briar, Rye

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Brigandiini (1 Panssari, _kookas_)
- Linko (d6)
- Käsikirves (d6)
- Karkote (valitse tyyppi, 3 käyttöä)

## Mikä sai sinusta yliotteen? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| ----- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | Ahnas tuholaisparvi, joka nielaisi niin sadon kuin eläimiäkin. Kun ei enää ollut mitään puolustettavaa, lähdit pois. Ota **Puhurisiemenuute** (3 käyttöä).  Sitä nauttimalla voit juosta nelinkertaisella nopeudella normaaliin vauhtiisi verrattuna. Sen jälkeen lisää kaksi **Väsymystä**.                                              |
| **2** | Sadon henki, joka suuttui vähäisistä kymmenyksistä. Tuli söi lähes kaiken, ja jälkikäteen sait kerättyä pussillisen **Tulisiemeniä** (d8, _räjähdys_, 4 käyttöä).                                                                                                                                                                         |
| **3** | Sarvipäinen, suurihampainen demoni, joka melkein tappoi sinut. Ota verestä tahriintunut **Luuveitsi** (d6). **Kriittisellä vahingolla** sen seuraava hyökkäys _vahvistuu_ kosketuksesta vereen.                                                                                                                                           |
| **4** | _Kuihtuminen_, Juuristosta peräisin oleva varsimätä. Ota **Tautinen vilja** (6 käyttöä), joka lakastuttaa nopeasti kaikki koskettamansa kasvit.                                                                                                                                                                                           |
| **5** | Sudet, tai niinhän sinä luulit. Olet nyt **Ihmissusi** [8 HP, 15 STR, 14 DEX, kynnet (d6+d6), puraisu (d8)]. WIL pysyy samana. Voit _muuttua_ halutessasi (kerran päivässä), mutta sinun on tehtävä WIL-pelastusheitto palataksesi takaisin. Kaikkien hyökkäyksistäsi eloonjääneiden on tehtävä WIL-pelastusheitto välttääkseen tartunta. |
| **6** | Sadonvarkaat. Kaikki eivät selvinneet hengissä, mutta jäit alakynteen. Aloita +d4 HP:lla ja **Teräväkärkisellä sapelilla** (d8).                                                                                                                                                                                                          |

## Mikä työkalu pelasti henkesi? Heitä 1d6:

|       |                        |                                                                                                                                                                                                          |
| ----- | ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Veriköynnöspiiska**  | d8 vahinkoa. **Kriittisellä vahingolla** se imee kohteesta verta, antaen aseen seuraavalle hyökkäykselle _räjähdys_-tyypin                                                                               |
| **2** | **Kolinanpitäjä**      | Käsikäyttöinen laite, joka päästää kovaa ääntä ja pelottaa useimmat olennot kauas pois.                                                                                                                  |
| **3** | **Aurinkotikku**       | Tarjoaa runsaasti lämpöä ja valoa jopa tunnin ajan. **Lataus**: Jätä voimakkaaseen auringonpaisteeseen koko päivän ajaksi. 1 käyttö.                                                                     |
| **4** | **Juuriside**          | Heitettäessä sitoo enintään suden kokoisen olennon maaperään kiinni lyhyeksi ajaksi.                                                                                                                     |
| **5** | **Viherpilli**         | Pieni huilu, joka rauhoittaa kasveja, helpottaen kulkua runsaskasvisten alueiden läpi.                                                                                                                   |
| **6** | **Ikikukkiva seppele** | Hiuspanta, jota koristavat kukat, jotka eivät koskaan kuihdu. **Kriittisen vahingon** osuessa kukat hajoavat pölyksi, mutta toimit kuten pelastusheittosi olisi onnistunut (STR-menetys tapahtuu silti). |
