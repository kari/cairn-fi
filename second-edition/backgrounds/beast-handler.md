# Pedonhoitaja

> Vain sinä voit kulkea villien olentojen keskellä ilman pelkoa ja tilanne hallinnassa. Sinulla on eläimiin yhteys, josta muut voivat vain uneksia... kunhan sinusta ei vain tule niiden suupalaa.

## Nimiä

Amara, Wulf, Mireille, Soren, Freki, Aster, Gerrik, Boreas, Veda, Matheus

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Nahkapiiska (d6)
- Tainnutusnuolet (STR-pelastusheitto tai nukahtaminen, 6 käyttöä)
- Houkutin
- Köysi (8 metriä)

## Mikä olento on erikoisalaasi? Heitä 1d6:

|       |                  |                                                                                                                                                                                                                                  |
| ----- | ---------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Hämähäkit**    | Ota **Nopea liekkisauva** ja **Öljykanisteri** (6 käyttöä). Ne voivat tuhota suuren hämähäkkipesän sekunneissa.                                                                                                                  |
| **2** | **Kissaeläimet** | Ota säkillinen **Viiksikarvanjuurta**. Sen tuoksu pystyy rauhoittamaan ja hillitsemään suurimmankin kissan.                                                                                                                      |
| **3** | **Koiraeläimet** | Ota seppele **Ukonhattua** ja **Suuri verkko**. Tehokas myös ihmissusia vastaan.                                                                                                                                                 |
| **4** | **Linnut**       | Ota **Viserryspilli** (3 käyttöä). Se voi jäljitellä mitä vain linnun kutsua, ja sitä voidaan käyttää myös yksinkertaisten viestien lähettämiseen. **Lataus**: Syötä linnunpoikasta aivan kuin olisit sen emo ja puhalla sitten. |
| **5** | **Jyrsijät**     | Ota **Panhuilu**, joka päästää korkean äänen, jonka vain jyrsijät kuulevat. Niin kauan kuin soitat, ne seuraavat sinua, jopa kuolemaansa.                                                                                        |
| **6** | **Käärmeet**     | Ota **Lämmittelykivi**, joka tuottaa vastustamatonta lämpöä, ja pullo **Vastamyrkkyä** (2 käyttöä).                                                                                                                              |

## Mitä olet oppinut luonnon olennoista? Heitä 1d6:

|       |                                                                                                                                                                                                           |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | Sen että maailmassa on paljon muutakin kuin mitä silmä näkee. Rauhallisella keskittymisellä voit lainata lähellä olevan **erityisalasi** olennon aisteja.                                                 |
| **2** | Sen että petojen käyttäytyminen on kieli itsessään. Tarkkaillessasi **erikoisalasi** petoja saat tietoa säätiloista ja uhkaavista katastrofeista.                                                         |
| **3** | Sen että metsästyksen syke on voimakkaan kiihkeä. Sinulla on tunne siitä, milloin saalistajat - myös ne joihin **et ole erikoistunut** - ovat lähellä.                                                    |
| **4** | Sen että maa on oma kielensä. Todennäköisyytesi eksyä maastoon, jota hallitsevat **erikoisalasi** pedot, pienenee yhdellä askeleella (esim. 4/6:sta tulee 3/6:sta).                                       |
| **5** | Sen että luonnon sinfonian voi kuulla, jos virittäytyy sen rytmiin. Kun ympärillä on **erikoisalasi** olentoja, ne voivat varoittaa sinua lähestyvästä vaarasta ennen sen saapumista.                     |
| **6** | Sen että selviytyminen on kyky sopeutua. Kerran päivässä voit ottaa yksinkertaisen ominaisuuden **erikoisalasi** olennolta (verkkomaiset sormet, pimeännäkö jne.). Lisää **Väsymys** jokaisella kerralla. |
