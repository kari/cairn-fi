# Vihertaituri

> Matkaa syvälle **Korpeen**, urkit sen salaisuuksia karheiden oksien ja kuiskaavien lehtien välistä. Et ole pelkkä vehreän valtakunnan tutkija vaan myös sen uskottu ystävä.

## Nimiä

Gunther, Moss, Fern, Lichen, Root, Willow, Sage, Yarrow, Rowan, Ash

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Rautaruukku
- Juuriveitsi (d6)
- Parantava voide (palauttaa 1d4 STR, 1 käyttö)
- Koristeellinen lankapallo (_mitätön_, _Vartiokuvio_ kerran päivässä) _Vartiokuvio_: Maahan ilmestyy hopeinen ympyrä, jonka läpimitta on 15 metriä. Valitse yksi laji, joka ei voi ylittää sitä.

## Miten Korpi on pettänyt sinut? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | Pahantuulinen metsänhenki kirosi sinut varkauden vuoksi ja leimasi sinut kaltaistensa viholliseksi. Ota **Besoaarikivi**. Sen nauttiminen parantaa minkä tahansa myrkyn (1 käyttö, ellei noudeta takaisin). |
| **2** | Läheinen ystävä katosi metsään. Nyt näet hänen kasvonsa aina keittämässäsi teessä. Ota **Tainnuttava keitos** (3 käyttöä).                                                                                  |
| **3** | Sait myrkytyksen ja menetit maku- ja hajuaistisi. Kestät nyt myrkyllisiä höyryjä ja sinulla on aina mukanasi **Vastamyrkkyä** (2 käyttöä).                                                                                  |
| **4** | Mullistavat kokeilusi muuttivat ihosi vihreäksi, ja saat nyt ravintoa kuin kasvi. Et tarvitse **muona-annoksia**, mutta päivä ilman riittävää auringonvaloa ja vettä jättää sinut _puutteeseen_.                                               |
| **5** | Vaikuttava ruumiskukkasi voitti paikallisen kilpailun ja tappoi sitten heti tuomarin. Pakenit, mutta et ilman **Palkintorahoja (100gp)** ja pidätysmääräystäsi.                                                             |
| **6** | Loit palauttavan rohdosuutteen, joka aiheuttaa myös vahingossa hedelmättömyyttä. Ota **parannusjuoma**, joka palauttaa STR:n kokonaan. Vain sinä tiedät sen tahattomat sivuvaikutukset.                        |

## Mikä pitää sinut turvassa Korvessa ollessasi? Heitä 1d6:

|       |                         |                                                                                                                                                           |
| ----- | ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Taula**               | Kirkkaanpunainen sieni, joka syttyy melko helposti tuleen. 3 käyttöä.                                                     |
| **2** | **Ritarinkannus**       | Hengitä veden alla jopa tunnin ajan. 1 käyttö, mutta voidaan jakaa pienempiin annoksiin.                                  |
| **3** | **Tahmea korsi**        | Puumainen ruoko, joka kovettuu pureskeltaessa liimaksi. 2 käyttöä.                                                        |
| **4** | **Virvatulilyhty**      | Takorautahäkissä, antaa hämärää valoa niin kauan kuin virvatuli pystyy ruokkimaan itseään lähellä olevalla kivulla ja pelolla.            |
| **5** | **Siemenpommi**         | Kangassäkki täynnä siemeniä, jotka räjähtävät osuessaan. d6 vahinkoa (_räjähdys_, 3 käyttöä).          |
| **6** | **Sarsaparillaköynnös** | Sitoo minkä tahansa aina hevosen kokoiseen paikalleen (STR-heitto vapautumiseen). Uudelleenkäytettävä. |
