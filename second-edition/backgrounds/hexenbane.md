# Noidansurma

> Olet pelkkä sormi oikeuden erehtymättömässä kädessä. Menet sinne minne muut eivät uskalla astua, periksiantamattomana ja rikkomattomana.

## Nimiä

Percival, Felix, Isolde, Wolfram, Aldric, Eira, Oswin, Ivor, Brunhilda, Beatrix

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Veljeskunnan vaatetus (_mitätön_)
- Siunatut rohdosuutteet
- Hopeaveitsi (d6)
- Varsijuosi (d8, _kookas_)

## Mihin veljeskuntaan kuulut? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                   |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | **Risteysten Veljeskunta**. Ota **Ley-linojen taskuetsin**. Se osoittaa läheisiin ley-linjoihin ja muihin salaperäisten voimien lähteisiin. Jos menetät sen, rangaistuksena on kuolema.                                                                                           |
| **2** | **Verestävän Tähden Veljeskunta**. Ota **Rautainen piikkinuija** (d8). Se loistaa heikosti pimeässä ja muuttuu hyvin kuumaksi noituuden läsnäollessa.                                                                                                                          |
| **3** | **Lasikuvion Veljeskunta**. Ota lyhytmiekka (d8) ja rengaspanssari (2 Panssaria, _kookas_). Sinulla on yhteyshenkilöitä useimmissa kylissä (mitä kauempana, sitä paremmin), jotka ovat valmiita tarjoamaan apua, ruokaa tai jopa aseita. |
| **4** | **Tyhjän Silmän Veljeskunta**. Ota **Tyhjyyslasin sirpale**. Kurkista sen läpi nähdäksesi näkymättömiä merkkejä, olentoja ja muita maagisia vaikutuksia. Menetät silmäsi käytön tunniksi sen jälkeen (olet _puutteessa_).                                      |
| **5** | **Ulvaan Veljeskunta**. Kerran päivässä voit muuttua sudeksi. Ota **Elohopeaketju**. Ilman sitä et pysty muuttumaan takaisin.                                                                                                                                                     |
| **6** | **Hiljaisen Harson Veljeskunta**. Ota säkkikankaaseen kääritty **Tukahduskivi** (2 käyttöä). Sammuttaa kaikki lähellä olevat liekit, kun se on kosketuksissa ilman kanssa.                                                                                                     |

## Mikä oli valasi? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | **Rehellisyys**. Valitse **asetyyppi** (tylppä, miekka jne.). Tämäntyyppiset sinuun kohdistuvat hyökkäykset ovat _heikentyneitä_. Jos valasi rikkoutuu, menetät d4 WILiä.                                                                                                                                                                                                                                             |
| **2** | **Köyhyys**. Sinulla on mukanasi _Irrottelu_-**loitsukirja**. Vain sinä voit käyttää sitä. Jos valasi rikkoutuu, se räjähtää (d12 STR-vahinkoa). Irrottelu: Mikä vain ruumiinosasi voidaan halutessasi irrottaa ja kiinnittää uudelleen, aiheuttamatta kipua tai vahinkoa. Voit silti hallita niitä. _Kaikki repeytyneet tai turmeltuneet sivut uusiutuvat takaisin._ |
| **3** | **Epäitsekkyys**. Olet immuuni **mieleen kohdistuville maagisille vaikutuksille**, kuten _lumous_, _vihanpito_, _raivo_ ja niin edelleen. Jos rikot tämän valan, menetät d6 WIL:iä.                                                                                                                                                                                                                                                                      |
| **4** | **Armo**. Valitse **asetyyppi** (tylppä, miekka jne.). Hyökkäykset tällaisella aseella ovat _vahvistettuja_. Jos valasi rikkoutuu, et voi enää koskaan käyttää kyseistä asetyyppiä.                                                                                                                                                                                                                                   |
| **5** | **Hyväntekeväisyys**. Kerran päivässä voit torjua yhden **Väsymyksen**. Jos valasi rikkoutuu, menetät pysyvästi yhden varustepaikan.                                                                                                                                                                                                                                                                                                                                     |
| **6** | **Urheus**. Kun aiheutat ensimmäistä kertaa **Kriittistä vahinkoa**, saat +d4 HP, palaten takaisin edelliseen rajaan taistelun päätyttyä. Jos valasi rikkoutuu, kuolet.                                                                                                                                                                                                                                                                                                  |
