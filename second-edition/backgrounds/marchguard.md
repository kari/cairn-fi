# Rajavartija

> Verivala sitoo partioimaan rajoja ja suojelemaan valtakuntaa. Kerran vannottua Valaa ei voi rikkoa. Kaarti löytää aina omansa.

## Nimiä

Gann, Light, Saoirse, Frost, Thorn, Reed, Dirk, Ragnar, Brie, Aasim

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Lyhty
- Öljykannu (6 käyttöä)
- Pitkämiekka (d10, _kookas_)
- Kovetetut nahkavaatteet (1 Panssari)

## Miksi vannoit Valan?

|       |                                                                                                                                                                                                                                                                       |
| ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **1** | Suvussasi on pitkät perinteet palveluksesta, ja sinut koulutettiin jo varhain selviytymään luonnossa. Kun teet Ruoan hankkiminen -toiminnon, saaliisi kasvaa yhdellä askeleella (esim. 1d4 > 1d6). |
| **2** | Tuomittuna vankina vala oli yksinkertaisesti vain keino välttää rangaistus. Ota **Tiirikat** ja **Avain** (_mitätön_) turvataloon.                                                                                 |
| **3** | Aatelissyntyisenä liityit mukaan paetaksesi perheongelmia. Ota **Hanhenuntuvahuopa** (mahtuu kaksi), jonka varastit ennen kotoa lähtöäsi.                                                                          |
| **4** | Kun perheesi menetti kaiken, vannoit Valan välttyäksesi joutumasta taakaksi. Ota ylimääräiset **Muona-annokset** (3 käyttöä) ja joukko **Heittoveitsiä** (d6).                                  |
| **5** | **Rajakaartin** jäsen pelasti henkesi ja se sai sinut liittymään heidän riveihinsä. Ota **Jalkanaru** ja **Luonnoskirja** täynnä yksityiskohtaisia piirustuksia.                                                                      |
| **6** | Olit synkässä paikassa ja päätit, että elämäsi kaipaa uuden suunnan. Et ole vieläkään varma, oliko tämä oikea valinta. Ota **Öljykangastakki** ja **Karttapaperia**.                                                  |

## Mitä sinulla on mukanasi todisteena Valastasi?

|       |                           |                                                                                                                                                                                                                                                                                                                                                  |
| ----- | ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | **Vaikuttava rintaneula** | Kaartin metallinen kunniamerkki. Se voi avata ovia, mutta jättää jäljen. _Mitätön_.                                                                                                                                                                                                              |
| **2** | **Valakompassi**          | Ei osoita kohti pohjoista, vaan lähimpään Kaartin jäseneen. Sen avulla tiedät myös, milloin he ovat lähellä.                                                                                                                                                                                                     |
| **3** | **Vetokivet**             | Kaksi sysimustaa kiveä. Kun kivet erotetaan toisistaan, ne vierivät aina toisiaan kohti.                                                                                                                                                                                                                         |
| **4** | **Palopullo**             | Voimakkaan alkoholipitoinen, silti oudon herkullista. Heitettäessä se luo 3 metriä korkean liekkiseinän, joka palaa loppuun muutaman minuutin kuluttua. 1 käyttö.                                                                                                                                |
| **5** | **Kivunrengas**           | Kosketa loukkaantunutta olentoa siirtääksesi sen haavat sinulle. (Vaihda heidän menettämänsä STR:nsä omaasi.) **Lataus**: Käytä sormusta ollessasi täysin terve. Menetät 1 STR-pisteen, pysyvästi. _Mitätön_. |
| **6** | **Salametsästäjän Murhe** | Voimakkaasti tuoksuvat nuolet (3 käyttöä). Tuoksu on tarpeeksi voimakas, jotta sitä voi seurata helposti.                                                                                                                                                                                     |
