# Vaanija

> Olet öinen haamu, ohikiitävä varjo, joka livahtaa saaliinsa luo sen huomaamatta. Jokainen tappo on oveluuden ja eläimellisen päättäväisyyden koitos, elämän ja kuoleman välinen taistelu. Tiedät, että jonain päivänä häviät. Odotat sitä innolla.

## Nimiä

Winda, Brielle, Theron, Chayse, Nuja, Dev, Raven, Arawan, Sable, Baruani

## Aloitusvälineet

- 3d6 kultarahaa
- Muona-annokset (3 käyttöä)
- Soihtu (3 käyttöä)
- Peite
- Kovetetut nahkavaatteet (1 Panssari)
- Lyhytmiekka (d6)
- Jousilaukeava loukku (4 STR-vahinkoa)

## Mitä metsästit viimeksi? Heitä 1d6:

|       |                                                                                                                                                                                                                                                                                                                                                                               |   |
| ----- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | - |
| **1** | _Valetulikärpästä_, joka houkutteli vedenkantajia hehkuvalla syötillään. Ota **Alkemistinen jalka** (d8, _mitätön_ kun sitä käytetään) sen repimän raajasi tilalle ja **Öljykannu** (6 käyttöä). Raaja on immuuni kuumuudelle ja myrkyille. Se pitää öljytä päivittäin. |   |
| **2** | _Jäänokkosta_, joka pyydysti ja imi lampaita kuiviin. Menetit palkkiosi, kun tuomasi sieni tappoi puolet laumasta. Ota **Kuuransiemen** (1 käyttö). Se jäädyttää minkä tahansa vesistön, riippumatta sen koosta. Älä syö sitä.                                             |   |
| **3** | Hopeista suoryömijää, joka tappoi jonkun läheisesi. Kannat nyt sen **hammasta** (_mitätön_) ketjussa kaulassasi ympärillä varoituksena muille sen kaltaisille. _Hammas humisee hiljaa, kun jokin väijyy sinua_.                                                                                            |   |
| **4** | Pahansuopaa _metsänhenkeä_, joka myrkytti kotitilan. Pelastit **Sydänsiemene** kuolevan puun juurista. (Istuta se kasvattaaksesi uusi metsä.) Ota myös **Rautaiset rannesuojukset** (+1 Panssari, _kookkaat_).                                                          |   |
| **5** | _Onttoa sutta_, joka oli pelotellut matkailijoita. Sinua kävi puoliksi nälkiintynyttä olentoa sääli ja hoidit sen takaisin kuntoon. Nyt se on sinulle uskollinen kuolemaan asti. Se on myös loistava tunnelinkaivaja. 5 HP, 11 STR, 13 DEX, 8 WIL, hampaat (d6).           |   |
| **6** | _Latvuskerttulia_. Sen sukusolut ovat erittäin arvokkaita, jos ne saadaan huolellisesti talteen. Onnistuit, mutta jätit sen pesän susien armoille. Ota **Kuorimaveitsi** (d6), **20gp ylimääräistä** ja kauhea katumuksen tunne.                                                           |   |

## Mikä työkalu on aina repussasi? Heitä 1d6:

|       |                     |                                                                                                                                                                                            |
| ----- | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **1** | **Pontikkaa**       | Pitää sinut lämpimänä hyvinä aikoina ja toimii räjähteenä pahassa paikassa. 3 Käyttöä.                                                                     |
| **2** | **Jäljenravistin**  | Meluisa soitin, joka paljastaa läheiset jäljet, vaikka ne olisivat syvällä piilossa.                                                                                       |
| **3** | **Univoide**        | Pala vahaa. Vedessä keitettynä sen höyry toimii tainnutukseen.                                                                                             |
| **4** | **Piikki ja naru**  | Vaikeassa maastossa liikkumiseen tai improvisoitujen ansojen ja rakennelmien tekemiseen.                                                                                   |
| **5** | **Rautahelistin**   | Meluisa laite, jolla voit hämätä tai pelotella saalistasi. Kuulostaa vakuuttavasti käärmeeltä.                                                             |
| **6** | **Kovettumisliima** | Tekee minkä tahansa litteän materiaalin (kangas, nahka, hiekka) kovaksi kuin kivi. Kallista (20gp pullo, 3 käyttöä). |
